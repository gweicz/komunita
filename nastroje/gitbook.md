# GitBook

* Organizace: [**app.gitbook.com/@gweicz**](https://app.gitbook.com/@gweicz) (přístupné jen pro editory)
* Spravuje: [Council](../council/)

Obsah z GitBooku je automaticky synchronizován s příslušným GitHub repozitářem.
